# Anthropic Cybersecurity Skills

A curated collection of cybersecurity skills and prompts for Claude AI, designed to assist security professionals, researchers, and learners with various cybersecurity tasks.

> **Fork of [mukul975/Anthropic-Cybersecurity-Skills](https://github.com/mukul975/Anthropic-Cybersecurity-Skills)**

## ⚠️ Disclaimer

This project is intended for **educational and legitimate security research purposes only**. All skills and prompts should be used responsibly and ethically. Do not use these tools for unauthorized access, malicious activities, or any illegal purposes.

## 📦 Installation

### Via Claude Plugin Marketplace

1. Open Claude and navigate to the Plugin Marketplace
2. Search for "Cybersecurity Skills"
3. Click **Install**

### Manual Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/Anthropic-Cybersecurity-Skills.git
   ```
2. Follow the [Claude plugin documentation](https://docs.anthropic.com) to load local plugins.

## 🛡️ Skill Categories

| Category | Description |
|---|---|
| **Reconnaissance** | OSINT, footprinting, and information gathering techniques |
| **Vulnerability Analysis** | CVE lookup, vulnerability scanning, and assessment |
| **Penetration Testing** | Ethical hacking methodologies and techniques |
| **Incident Response** | Threat detection, containment, and remediation |
| **Malware Analysis** | Static and dynamic analysis techniques |
| **Cryptography** | Encryption, hashing, and cryptographic concepts |
| **Network Security** | Firewall rules, IDS/IPS, and network hardening |
| **Web Security** | OWASP Top 10, XSS, SQLi, and web app security |
| **Forensics** | Digital forensics and evidence collection |
| **CTF Assistance** | Capture the Flag challenge guidance |

## 🚀 Usage

Once installed, you can invoke cybersecurity skills in Claude by describing your task:

```
"Help me analyze this suspicious network traffic log for indicators of compromise."

"Explain the MITRE ATT&CK technique T1059 and how to detect it."

"Review this Python script for common security vulnerabilities."
```
